# LineFollowerBot
 Line Follower Robot using Arduino Uno Based on PID Algorithm
## Parts Required :
- Chasis
- Arduino Uno Microcontroller
- Motors + Wheels 
- L298N motor driver board
- 5 TCRT5000 infrared Sensors
- LEDs + resistors
- One Breadboard for LED connections
- Batteries
- Cables, screws and bolts
